# 小红书图文生成器

一个小型 Web 应用：批量导入背景图和文案，并把居中文本渲染到每张图上（每行 7 个字符），字体随机。支持内置模板、多图上传和 ZIP 导出。

## 功能
- 批量上传图片、批量输入文案（每行对应一张图）。
- 内置模板画廊，支持全选 / 清空 / 多选。
- 从可配置字体池中为每张图随机字体。
- 居中排版，自动换行（每行 7 个字符）。
- 英文单词保持完整，不会被拆分。
- 一键 ZIP 打包导出所有生成图片。

## 快速开始
1. 用浏览器打开 `index.html`。
2. 选择模板（可选）和/或上传背景图。
3. 在文本框粘贴文案（每行对应一张图）。
4. 点击 “Generate Preview”。
5. 点击 “ZIP Export” 下载全部结果。

## 匹配规则
- 先匹配模板，再匹配上传图片。
- 文案按顺序逐行匹配。
- 文案行数少于图片数量时，多余图片不生成文本。

## 文本规则
- 每 7 个字符换行。
- 英文单词（A-Z、a-z、0-9、连字符、撇号）保持完整。
- 单个英文单词超过 7 个字符时，独占一行。

## 字体
- 字体通过 `index.html` 中的 Google Fonts 加载。
- 在 `app.js` 中修改字体池以添加或移除字体。
- 网络不可用时，浏览器会回退到系统字体。

## 模板
内置模板位于 `templates/` 目录。模板画廊配置在 `app.js`（见 `templateLibrary`）。

添加新模板：
1. 把图片放到 `templates/`。
2. 在 `app.js` 的 `templateLibrary` 中新增条目。

## 项目结构
- `index.html`: UI 布局
- `styles.css`: 样式
- `app.js`: 业务逻辑（渲染、批量处理、ZIP 导出）
- `templates/`: 内置背景模板

## 说明
- ZIP 导出使用 CDN 上的 JSZip（需要联网）。
- 若需完全离线使用，请本地起一个静态服务器，并将 CDN 资源替换为本地文件。
